# 视觉风格

## 设计气质

商户端应呈现“清爽、可信、服务导向、SaaS 工具感”。它不是营销官网，也不是内部后台。画面要能承载订单、税号、申报、购买等复杂业务，但用户看到后不应产生“我是不是进了运营后台”的错觉。

## 色彩

| 类型 | 色值 | 用途 |
| --- | --- | --- |
| 品牌主蓝 | `#0077FF` | 主按钮、链接、当前菜单、当前 tab、步骤完成态 |
| 主文本 | `#333333` | 标题、表格主内容、关键字段 |
| 次文本 | `#666666` | 说明、字段值、普通辅助信息 |
| 弱文本 | `#999999` | 占位符、弱提示、版权、次级说明 |
| 页面背景 | `#F5F7FA` / `#F7F8FA` | 主应用背景 |
| 卡片背景 | `#FFFFFF` | 内容容器、筛选区、表格区、商品卡 |
| 分割/边框 | `#E7E7E7` / `#E8E8E8` | 表格线、输入框边框、卡片分隔 |
| 价格/危险 | `#F5222D` / `#FF0000` | 价格、欠费、驳回、危险信息 |
| 成功/通过 | `#18B56A` / `#00B578` | 已授权、完成、通过类标签 |
| 警示黄 | `#FFEB3B` | UAT 沙箱警示条，正式环境慎用 |
| 商品促销橙红 | `#E64B44` | 优惠、券、促销入口 |

使用规则：

- 蓝色只承担主路径，不要把普通文本也刷蓝。
- 红色主要用于价格、驳回、危险、必读警示。
- 大面积背景保持浅灰和白色，避免后台深色侧栏风格。
- 卡片阴影要轻，边界清楚即可，不要做浮夸投影。

## 字体与字号

- 中文优先：`PingFang SC`, `Microsoft YaHei`, `Helvetica Neue`, Arial, sans-serif。
- 正文：14px，行高约 20-22px。
- 页面标题/区块标题：16px，字重 500-600。
- 重要业务标题：18px，字重 500-600。
- 辅助说明/标签：12px。
- 金额可使用 16-22px，根据页面权重加粗或用红色强调。

不要使用负字距，不要用大标题堆排场。商户端大多数页面是任务界面，不是发布会主视觉。

## 布局骨架

- 左侧导航宽度约 180px，白底，Logo 顶部固定。
- 顶部栏高度约 44px，白底，右侧放购物车、邀请有礼、优惠专区、帮助、用户信息。
- 主内容区左边距与侧栏衔接，背景浅灰。
- 主内容卡片左右留白约 16px，卡片内边距约 16px。
- 卡片圆角约 8px，边框和阴影都要克制。
- 右侧浮动工具条贴近内容右侧，包含邀请有礼、购物车、客服、公众号。
- 底部 UAT 黄色警示条固定在视口底部，仅用于测试环境。

## 间距

- 页面主内容边距：16px。
- 卡片间距：16px。
- 表单项横向间距：8-12px。
- 表格单元格左右内边距：12px。
- 商品卡片间距：12-16px。
- 登录页表单项垂直间距：24-30px。

## 图像与插画

- 登录页可使用左侧营销/促销图，右侧登录表单保持克制。
- 商品卡使用国旗、服务缩略图、品牌水印、售罄遮罩等业务图形。
- 404 使用轻量插画，配合清晰文字和返回按钮。
- 不要用纯装饰渐变球、抽象背景糊弄业务页面。
