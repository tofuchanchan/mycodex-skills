# OST系统组件规范

## 按钮组件

### 主按钮
- **样式**: 蓝色背景(#3b82f6)，白色文字，圆角8px
- **尺寸**: 高度36px，内边距 16px 24px
- **状态**:
  - 默认: 蓝色背景
  - 悬停: 深蓝(#2563eb)
  - 点击: 更深蓝(#1d4ed8)
  - 禁用: 灰色背景(#d9d9d9)，灰色文字

### 次按钮
- **样式**: 白色背景，蓝色边框(#3b82f6)，蓝色文字，圆角8px
- **尺寸**: 高度36px，内边距 16px 24px
- **状态**:
  - 默认: 白色背景，蓝色边框
  - 悬停: 浅蓝色背景(#eff6ff)
  - 禁用: 灰色边框，灰色文字

### 文字按钮
- **样式**: 无背景，蓝色文字，下划线可选
- **尺寸**: 高度32px
- **用途**: 次要操作、链接跳转

### 图标按钮
- **样式**: 圆形或方形，图标居中
- **尺寸**: 36px x 36px
- **用途**: 紧凑空间的操作按钮

## 输入框组件

### 文本输入框
- **样式**: 白色背景，浅灰边框(#e8e8e8)，圆角8px
- **尺寸**: 高度36px，内边距 0 12px
- **状态**:
  - 默认: 浅灰边框
  - 聚焦: 蓝色边框(#3b82f6)，蓝色阴影
  - 错误: 红色边框(#f5222d)
  - 禁用: 灰色背景(#f5f5f5)

### 下拉选择框
- **样式**: 白色背景，浅灰边框，右侧下拉箭头
- **尺寸**: 高度36px，内边距 0 12px
- **展开**: 下拉菜单覆盖下方内容

### 日期选择框
- **样式**: 同文本输入框，右侧日历图标
- **尺寸**: 高度36px

### 多选框/单选框
- **样式**: 圆形/方形边框，选中时填充蓝色
- **尺寸**: 16px x 16px
- **标签**: 右侧文字，14px

## 卡片组件

### 基础卡片
- **样式**: 白色背景，浅灰边框(#e8e8e8)，圆角8px，轻微阴影
- **内边距**: 16px
- **阴影**: box-shadow: 0 2px 8px rgba(0,0,0,0.06)

### 带标题卡片
- **结构**: 标题区 + 内容区
- **标题**: 16px，加粗，底部边框分隔
- **内容**: 14px正文

## 表格组件

### 基础表格
- **样式**: 白色背景，全边框
- **表头**: 浅灰背景(#fafafa)，文字加粗
- **行高**: 40px
- **悬停**: 浅蓝背景(#f0f5ff)
- **斑马线**: 可选，奇数/偶数行不同背景

### 表格操作列
- **按钮组**: 多个图标按钮横向排列
- **间距**: 8px
- **操作类型**: 查看详情、编辑、删除、更多操作

### 表格工具栏
- **位置**: 表格上方
- **左侧操作**: 批量选择复选框、批量操作按钮
- **右侧操作**: 导出数据、字段设置、筛选设置
- **下拉选择**: 批量操作下拉菜单

### 分页组件
- **位置**: 表格底部
- **组件**: 首页、上一页、页码、下一页、末页
- **附加信息**: 总条数、当前页/总页数

### 表格列设置
- **功能**: 可配置显示/隐藏列
- **触发**: 字段设置按钮
- **形式**: 弹窗或下拉菜单

## 弹窗组件

### 基础弹窗
- **样式**: 白色背景，圆角8px，阴影效果
- **宽度**: 固定宽度(520px)或自适应
- **遮罩**: 半透明黑色(#000000, 40%)

### 弹窗结构
- **标题栏**: 16px加粗文字，右侧关闭按钮
- **内容区**: 14px正文，内边距24px
- **按钮区**: 底部，确认/取消按钮右对齐

## 标签页组件

### 基础标签页
- **样式**: 顶部标签栏，下划线指示当前选中
- **标签间距**: 24px
- **选中状态**: 蓝色下划线，蓝色文字
- **未选中状态**: 灰色文字

### 标签页结构
- **标签区域**: 页面内容区域顶部
- **内容区域**: 标签页下方，卡片式布局

## 搜索表单组件

### 表单布局
- **多列布局**: 2-4列并排
- **标签位置**: 输入框上方或左侧
- **字段间距**: 水平16px，垂直12px

### 表单控件组合
- **下拉选择框**: 带请选择提示文字
- **文本输入框**: 带占位符提示
- **日期选择框**: 只读输入框，点击弹出日期选择器
- **范围选择**: 开始日期 + 结束日期组合

### 操作按钮组
- **查询按钮**: 主按钮样式
- **重置按钮**: 次按钮样式
- **导出按钮**: 带图标按钮

## 标签组件

### 状态标签
- **样式**: 圆角4px，内边距 4px 12px
- **颜色**:
  - 成功: 绿色背景，白色文字
  - 警告: 橙色背景，白色文字
  - 危险: 红色背景，白色文字
  - 信息: 蓝色背景，白色文字

### 普通标签
- **样式**: 白色背景，灰色边框，圆角4px
- **文字**: 12px灰色

## 进度条组件

### 线性进度条
- **样式**: 灰色背景，蓝色进度
- **高度**: 6px
- **圆角**: 3px

### 环形进度条
- **尺寸**: 根据场景调整
- **颜色**: 蓝色强调色

## 导航菜单组件

### 侧边栏菜单
- **样式**: 深蓝色背景(#1a2744)
- **宽度**: 200px
- **菜单项**: 40px高度，图标+文字
- **选中状态**: 蓝色高亮(#3b82f6)

### 面包屑导航
- **样式**: 灰色文字，箭头分隔
- **当前项**: 黑色加粗

## 提示组件

### 提示框(Toast)
- **位置**: 顶部居中或右下角
- **样式**: 圆角8px，阴影效果
- **自动消失**: 3秒后自动关闭

### 工具提示(Tooltip)
- **样式**: 白色背景，阴影，三角指向目标
- **触发**: 悬停显示

## 表单验证

### 错误提示
- **样式**: 红色文字(#f5222d)，12px
- **位置**: 输入框下方
- **图标**: 红色错误图标前置
