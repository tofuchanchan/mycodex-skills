## 颜色使用规则

1. **主色优先**: 主要操作按钮、重要链接使用蓝色强调色
2. **状态明确**: 使用状态颜色时确保语义清晰
3. **对比度**: 文字与背景对比度至少4.5:1
4. **一致性**: 相同类型元素使用统一颜色
