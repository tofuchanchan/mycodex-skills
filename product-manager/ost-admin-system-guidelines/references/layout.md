# OST系统布局规范

## 整体布局

### 页面结构
```
┌─────────────────────────────────────────────────────────────────────┐
│  Top Bar (高度: 60px)                                              │
│  [Logo] [搜索框] [用户信息] [操作按钮]                               │
├─────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────┐  ┌──────────────────────────────────────────┐ │
│  │ Sidebar         │  │ Content Area                            │ │
│  │ (宽度: 200px)   │  │ (自适应剩余宽度)                          │ │
│  │ ┌─────────────┐ │  │                                          │ │
│  │ │ 菜单组1      │ │  │  ┌──────────────────────────────────┐  │ │
│  │ │ 菜单组2      │ │  │  │       卡片/表单区域               │  │ │
│  │ │ ...          │ │  │  │                                  │  │ │
│  │ └─────────────┘ │  │  │                                  │  │ │
│  └─────────────────┘  │  └──────────────────────────────────┘  │ │
│                       │                                          │ │
│                       │  ┌──────────────────────────────────┐  │ │
│                       │  │       表格/列表区域               │  │ │
│                       │  └──────────────────────────────────┘  │ │
│                       └──────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

### 布局参数
| 区域 | 尺寸 | 说明 |
|------|------|------|
| 顶部导航栏 | 高度60px | 固定定位，始终显示 |
| 侧边栏 | 宽度200px | 固定宽度，可折叠 |
| 内容区域 | 自适应 | 剩余宽度 |
| 页面边距 | 16px | 内容区与边界距离 |

## 网格系统

### 基础网格
- **列数**: 12列
- **间距**: 16px（gutter）
- **响应断点**:
  - 桌面端: ≥1200px
  - 平板端: 768px-1199px
  - 移动端: <768px

### 常用布局模式

#### 两栏布局
```
┌─────────────────────────────────────────────────────────────┐
│  ┌──────────────┐  ┌────────────────────────────────────┐  │
│  │  左侧栏      │  │           主内容区                   │  │
│  │  (3-4列)     │  │           (8-9列)                   │  │
│  └──────────────┘  └────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

#### 三栏布局
```
┌─────────────────────────────────────────────────────────────┐
│  ┌──────┐  ┌──────────────────┐  ┌────────────────────┐    │
│  │边栏  │  │      主内容       │  │    右侧边栏        │    │
│  │(2列) │  │     (8列)         │  │     (2列)          │    │
│  └──────┘  └──────────────────┘  └────────────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

## 卡片布局

### 卡片间距
- **水平间距**: 16px
- **垂直间距**: 16px
- **卡片内边距**: 16px

### 卡片排列
- **单行单列**: 独立卡片
- **双列布局**: 并排两个卡片
- **栅格布局**: 多卡片网格排列

## 表单布局

### 表单结构
- **标签位置**: 左对齐或顶部对齐
- **标签宽度**: 120px（左对齐）
- **输入框宽度**: 200px-300px
- **字段间距**: 16px（垂直）

### 表单分组
- **分组标题**: 14px加粗
- **分组内边距**: 12px
- **分组边框**: 可选

## 表格布局

### 表格尺寸
- **最小宽度**: 自适应内容
- **最大宽度**: 100%容器宽度
- **单元格内边距**: 8px 12px

### 响应式表格
- **水平滚动**: 表格过长时显示滚动条
- **固定列**: 首列或关键列固定

## 弹窗布局

### 弹窗尺寸
- **默认宽度**: 520px
- **最小宽度**: 320px
- **最大宽度**: 90%视口宽度
- **最大高度**: 80%视口高度（带滚动）

### 弹窗内边距
- **标题区**: 16px 24px
- **内容区**: 24px
- **按钮区**: 16px 24px

## 间距系统

### 间距层级
| 间距 | 尺寸 | 用途 |
|------|------|------|
| xs | 4px | 元素内部小间距 |
| sm | 8px | 元素间距 |
| md | 12px | 组件间距 |
| lg | 16px | 卡片间距、页面边距 |
| xl | 24px | 大间距 |
| 2xl | 32px | 区域间距 |

## 对齐规则

### 文字对齐
- **标题**: 左对齐
- **正文**: 左对齐
- **表格内容**: 左对齐（数值右对齐）
- **按钮文字**: 居中对齐

### 元素对齐
- **表单标签**: 右对齐（左对齐布局时）
- **按钮组**: 右对齐（在容器内）
- **卡片内容**: 顶部对齐

## 响应式适配

### 桌面端 (≥1200px)
- 完整布局：侧边栏 + 内容区
- 多栏卡片布局

### 平板端 (768px-1199px)
- 侧边栏可折叠
- 减少列数

### 移动端 (<768px)
- 侧边栏隐藏（通过菜单按钮展开）
- 单列布局
- 按钮宽度100%

## 使用规则

1. **一致性**: 相同类型页面使用统一布局
2. **留白适度**: 避免内容过于拥挤
3. **响应优先**: 考虑不同屏幕尺寸
4. **层级分明**: 通过间距区分内容层级
