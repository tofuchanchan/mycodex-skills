# OST系统字体排版规范

## 字体选择

### 中文字体
- **首选**: Microsoft YaHei（微软雅黑）
- **备选**: PingFang SC（苹方）
- **降级**: SimHei（黑体）

### 英文字体
- **首选**: Arial
- **备选**: Helvetica
- **降级**: sans-serif

## 字号层级

### 标题层级
| 级别 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 20px | 600 | 1.5 | 页面主标题 |
| H2 | 18px | 600 | 1.5 | 区域标题 |
| H3 | 16px | 600 | 1.4 | 模块标题 |
| H4 | 14px | 600 | 1.4 | 卡片标题 |

### 正文字体
| 类型 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 正文 | 14px | 400 | 1.5 | 主要内容文字 |
| 小正文 | 13px | 400 | 1.5 | 次要内容 |
| 辅助文字 | 12px | 400 | 1.4 | 提示、说明、标签 |

### 特殊用途
| 类型 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 按钮文字 | 14px | 500 | 按钮标签 |
| 表格内容 | 13px | 400 | 表格单元格 |
| 表单标签 | 14px | 500 | 表单字段标签 |
| 链接文字 | 14px | 400 | 超链接 |

## 字重规范

| 字重值 | 描述 | 用途 |
|--------|------|------|
| 400 | 常规 (Normal) | 正文、辅助文字 |
| 500 | 中等 (Medium) | 按钮、表单标签 |
| 600 | 半粗 (Semi-Bold) | 标题、重要文字 |

## 行高规范

| 内容类型 | 行高 |
|----------|------|
| 标题 | 1.5 |
| 正文 | 1.5 |
| 表格 | 1.4 |
| 按钮 | 1.4 |

## 字间距

- **默认**: 0
- **标题**: 0.5px
- **按钮**: 1px

## 使用规则

1. **层级清晰**: 使用明确的标题层级区分内容重要性
2. **一致性**: 相同类型内容使用统一字号和字重
3. **可读性**: 大段文字使用合适行高（1.5-1.8）
4. **响应式**: 在不同屏幕尺寸上保持可读性
