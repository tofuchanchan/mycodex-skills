# OST系统交互规范

## 按钮交互

### 按钮状态
| 状态 | 视觉变化 | 交互行为 |
|------|---------|---------|
| 默认 | 正常样式 | 无 |
| 悬停 | 背景色加深/变亮 | 鼠标指针变为手型 |
| 按下 | 轻微缩放(98%) | 触发点击事件 |
| 禁用 | 灰色化，降低透明度 | 禁止点击 |

### 按钮反馈
- **点击反馈**: 按钮短暂变暗或缩放
- **加载状态**: 显示加载图标，禁用点击

## 表单交互

### 输入框状态
| 状态 | 视觉变化 | 交互行为 |
|------|---------|---------|
| 默认 | 浅灰边框 | 无 |
| 聚焦 | 蓝色边框，蓝色阴影 | 显示光标，准备输入 |
| 错误 | 红色边框 | 显示错误提示 |
| 成功 | 绿色边框 | 显示成功提示 |

### 表单验证
- **实时验证**: 输入时即时检查
- **失焦验证**: 离开字段时验证
- **提交验证**: 提交前全面验证

### 错误提示
- **显示位置**: 输入框下方
- **显示时机**: 失焦后或提交时
- **消失时机**: 输入正确后

## 表格交互

### 行交互
| 操作 | 行为 | 视觉反馈 |
|------|------|---------|
| 悬停 | 高亮显示 | 背景色变浅 |
| 点击 | 触发行详情或选中 | 选中状态高亮 |
| 双击 | 查看详情弹窗 | 打开弹窗 |

### 表格操作
- **批量选择**: 复选框列，全选功能
- **分页导航**: 底部分页控件
- **排序**: 点击表头排序

## 弹窗交互

### 弹窗控制
| 操作 | 行为 |
|------|------|
| 点击关闭按钮 | 关闭弹窗 |
| 点击遮罩层 | 关闭弹窗 |
| 按ESC键 | 关闭弹窗 |

### 弹窗动画
- **打开动画**: 淡入 + 缩放
- **关闭动画**: 淡出 + 缩小

## 导航交互

### 菜单交互
| 操作 | 行为 | 视觉反馈 |
|------|------|---------|
| 点击菜单项 | 展开/折叠子菜单 | 箭头旋转 |
| 点击链接项 | 跳转到对应页面 | 选中高亮 |
| 悬停 | 显示子菜单（如果有） | 背景色变化 |

### 面包屑交互
- **点击项**: 跳转到对应层级
- **悬停**: 显示完整路径提示

## 加载状态

### 加载指示器
- **页面加载**: 全屏加载动画
- **区域加载**: 局部加载动画
- **按钮加载**: 按钮内加载图标

### 加载动画类型
- **旋转图标**: 圆形旋转动画
- **进度条**: 线性进度指示
- **骨架屏**: 占位骨架

## 提示反馈

### Toast提示
| 类型 | 颜色 | 用途 |
|------|------|------|
| 成功 | 绿色 | 操作成功提示 |
| 警告 | 橙色 | 警告信息提示 |
| 错误 | 红色 | 错误信息提示 |
| 信息 | 蓝色 | 一般信息提示 |

### Toast行为
- **显示位置**: 顶部居中或右下角
- **显示时长**: 3秒
- **关闭方式**: 自动关闭或点击关闭

### Tooltip提示
- **触发方式**: 悬停显示
- **显示位置**: 鼠标附近
- **消失方式**: 鼠标移开

## 拖拽交互

### 文件上传
- **拖拽区域**: 虚线边框，提示文字
- **拖拽状态**: 边框变蓝，背景变浅
- **放置效果**: 显示上传进度

## 滚动交互

### 列表滚动
- **虚拟滚动**: 长列表优化
- **滚动加载**: 滚动到底部自动加载更多
- **回到顶部**: 滚动一定距离后显示按钮

## 快捷键交互

### 全局快捷键
| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+F | 搜索 |
| ESC | 关闭弹窗/取消 |

### 表单快捷键
| 快捷键 | 功能 |
|--------|------|
| Enter | 提交表单 |
| Tab | 切换字段 |
| Shift+Tab | 反向切换 |

## 微交互动画

### 过渡动画
- **页面切换**: 淡入淡出
- **模态框**: 缩放+淡入
- **侧边栏**: 滑入滑出

### 状态变化动画
- **展开/折叠**: 平滑过渡
- **进度条**: 平滑增长
- **数值变化**: 数字动画

## 使用规则

1. **即时反馈**: 用户操作后立即给出视觉反馈
2. **操作可逆**: 重要操作提供确认或撤销机制
3. **状态明确**: 当前状态清晰可辨
4. **性能优先**: 动画不影响性能
